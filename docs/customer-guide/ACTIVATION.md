# License 激活指南 — V18 / 明证台

## 概述

V18 采用**机器绑定**的 License 机制。每个 License 只能在一台电脑上激活。激活后：
- 离线运行（不需联网验证）
- 30 天试用 → 永久激活（输入正式 License Key）
- 跨电脑迁移：先在旧电脑"反激活"，再在新电脑激活

## 首次激活

### 1. 获取 License Key

购买后，您会收到：
- **License Key** (形如 `V18-XXXX-XXXX-XXXX-XXXX`)
- **激活说明**邮件

如果还没购买，参见 [../../README.md](../../README.md) 或访问官网。

### 2. 启动 V18

双击桌面图标，或：
- 资源管理器 → 打开 V18 安装目录
- 双击 `launch_app.py`（如果已装 Python）
- 或双击 `run_gui.bat`（Windows 批处理启动器）

### 3. 激活 License

应用启动后，会弹出"激活"窗口：

```
┌─────────────────────────────┐
│   V18 License 激活            │
├─────────────────────────────┤
│ License Key: [_____________] │
│ Email:       [_____________] │
│                             │
│ [激活]  [试用 30 天]  [退出]  │
└─────────────────────────────┘
```

- **正式激活**: 粘贴 License Key + 邮箱 → 点击"激活"
- **试用 30 天**: 点击"试用 30 天"（无需 Key）

激活成功后会显示"激活成功"提示，并显示剩余天数。

### 4. 验证激活

激活后，V18 会在 `~/.v18/license.json` 写入：
```json
{
  "license_key": "V18-XXXX-XXXX-XXXX-XXXX",
  "machine_id": "abc123def456",
  "activated_at": "2026-06-25T10:00:00",
  "expires_at": "2027-06-25T10:00:00",
  "seats": 1
}
```

可在 V18 主界面 → 设置 → License 查看完整信息。

## 常见问题

### Q: License Key 写错了怎么办？
A: 重新粘贴正确的 Key 即可。激活失败会显示具体错误（"Key 无效" / "Key 已被使用"）。

### Q: 电脑重装系统后 License 失效了？
A: 电脑重装会改变 machine_id，原 License 失效。请联系 V18 销售，提供原 License Key + 新 machine_id，**免费重置一次**（每 License 每年 1 次免费重置）。

### Q: 想在多台电脑上用？
A: 每个 License 绑定一台电脑。需要在多台电脑用，请购买多个 License（或选择"小型律所版"包含 5 席位）。

### Q: 试用到期了，正式 Key 还没到？
A: 联系销售延长试用期。最多可延长 14 天（一次性）。

### Q: License 即将到期怎么续费？
A: 提前 30 天联系销售，续费后会自动延长（无需重新激活）。

## License 迁移

需要换电脑时：

1. **旧电脑**: 启动 V18 → 设置 → License → 点击"反激活"
   - 需要联网（首次反激活会校验服务器）
   - 反激活后原电脑 V18 立即失效
2. **新电脑**: 正常安装 + 激活流程，输入相同 License Key

⚠️ **注意**: 反激活不可撤销，反激活前请确认新电脑已就绪。

## 故障排查

### 激活时提示"机器 ID 已绑定其他 License"
说明这台电脑的 machine_id 已经激活过另一个 License（可能是您之前用过试用版，或别人用过）。
解决：
- 试用 → 试用到期 → 想激活正式版：请联系销售手动解绑
- 误用别人 License：请联系销售说明情况

### 激活时提示"无法连接到激活服务器"
激活需要联网（仅首次激活）。如果网络正常但仍报此错：
- 检查防火墙是否阻止 V18
- 检查是否在公司内网需要代理
- 临时方案：用 `V18_LICENSE_OFFLINE=1` 环境变量启用离线激活（需要销售提供离线 Key 文件）

### 激活后启动提示"License 已失效"
- 电脑时间被回拨到激活前 → 校准时间
- 电脑硬件变更导致 machine_id 变化 → 联系销售重置
- License 已过期 → 续费
