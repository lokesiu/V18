## 快速开始（30 秒）

1. **激活**: 双击 `launch_app.py` 或运行 `run_gui.bat`，按提示输入 License Key。
2. **导入材料**: 主页 → 上传 → 拖入 PDF / DOCX / 图片
3. **设置目标**: 选择案件类型 + 处理目标（"诉前保全"、"起诉书"等）
4. **运行**: 点击"开始分析"，9 阶段流水线自动跑（2-10 分钟）
5. **交付**: 完成后在交付物页签下载 DOCX / ZIP 包
