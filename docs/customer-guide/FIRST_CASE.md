# 跑通第一个案件 — V18 / 明证台

> 预计耗时: 10-15 分钟（含材料准备）

## 1. 准备材料

V18 接受以下材料类型：

| 类型 | 格式 | 大小限制 |
|------|------|----------|
| 文书 | PDF, DOCX, DOC | 单文件 ≤ 50 MB |
| 图片 | PNG, JPG | 单文件 ≤ 20 MB |
| 录音 | MP3, WAV, M4A | 单文件 ≤ 100 MB（自动转录） |
| 录像 | MP4, MOV | 单文件 ≤ 500 MB（仅企业版） |
| 聊天记录 | 微信导出 (.txt) | 任意 |

📌 **建议**:
- 文件名用案件名 + 日期（`张三借贷纠纷_起诉状.pdf`）
- PDF 比图片优先（OCR 准确率更高）
- 多文件可一次拖入

## 2. 创建新案件

1. 主界面 → 点击 **"新案件"** 按钮
2. 填写：
   - **案件名称**: 自由命名（例: "张三 vs 李四 民间借贷"）
   - **案件类型**: 下拉选择（民间借贷 / 婚姻家庭 / 交通事故 / 劳动争议 / 合同纠纷 / 知识产权 / 其他）
   - **当事人**:
     - 原告: 姓名 + 身份证 + 联系电话（可选）
     - 被告: 同上
   - **处理目标**: 多选（"诉前财产保全" / "起诉书" / "答辩状" / "律师函" / "证据清单"）
3. 点击 **"下一步"** → 进入材料上传页

## 3. 上传材料

3 种方式：
- **拖拽**: 把文件拖到上传区域
- **点击**: 点上传区域 → 文件选择对话框
- **批量**: 选中多个文件一次上传

上传后，V18 会自动：
- 解析文件类型
- 提取文本（PDF/DOCX）
- OCR 图片（如果需要）
- 转录音频（如果启用）

⏱ 等待时间取决于文件大小：典型 1 个 PDF 案件 1-3 分钟。

## 4. 运行 9 阶段流水线

材料准备好后，点击 **"开始分析"** 按钮。

V18 会按顺序跑 9 个阶段：
1. **接案** — 整理材料、识别当事人
2. **事实抽取** (DeepSeek) — 从非结构化材料中识别事实
3. **事实审查** (MiMo) — 验证事实真实性
4. **事实蒸馏** — 转为结构化卡片
5. **策略生成** (DeepSeek) — 生成诉讼策略
6. **策略复核** (MiMo) — 审查策略风险
7. **最终蒸馏** — 准备渲染
8. **渲染** — 生成 DOCX/PDF/XLSX/ZIP
9. **质量门禁** — 5 道检查

⏱ 整流程：2-10 分钟。可以在后台跑，V18 会推送通知。

📌 **阶段失败**: 任何阶段失败都会暂停流水线，并在 UI 中标红。鼠标悬停看错误详情。

## 5. 查看交付物

完成后，案件详情页会列出所有交付物：

| 交付物 | 格式 | 大小（典型） |
|--------|------|------------|
| 民事起诉状 | DOCX | 30-80 KB |
| 诉前财产保全申请书 | DOCX | 20-50 KB |
| 证据清单 | XLSX | 10-30 KB |
| 律师函 | DOCX | 15-40 KB |
| 完整交付包 | ZIP | 100-300 KB |

操作：
- **预览**: 点击文件 → 内置预览
- **下载**: 点击"下载"按钮
- **编辑**: 用 Word 打开 → 编辑 → 保存（V18 不感知你的修改）
- **打印**: 直接打印

## 6. 案例：从原始材料到起诉书

### 起始材料
- 1 份《借款合同》PDF
- 3 张银行转账截图
- 1 段微信聊天记录截图
- 1 张身份证（原告）

### 跑完后 V18 输出
- 民事起诉状（DOCX）: 完整格式，含原告诉求、事实理由、证据清单
- 证据清单（XLSX）: 每条证据的来源、证明目的、页码
- 诉前财产保全申请书: 标准格式，自动填入当事人信息
- 律师函: 发送给被告律师的催告函

### 律师需要做的
1. 下载起诉状 → Word 打开
2. **专业审核** AI 生成的内容（事实是否准确、诉求是否合理）
3. 根据需要微调（V18 提供模板，但案件细节需要律师判断）
4. 律师签字
5. 提交法院

💡 **专家技巧**: 第一次跑 V18 时，建议用**简单的借贷/合同纠纷**案件尝试。等熟悉流程后再用复杂案件（婚姻家庭、知识产权）。

## 7. 常见问题

### Q: 跑流水线时 V18 崩溃了
A: 检查错误日志（设置 → 查看日志）。常见原因：材料过大、AI 服务中断。重新跑即可。

### Q: 流水跑到一半卡住
A: 长时间（>10 分钟）无进度时：
1. 检查网络（AI 调用需要）
2. 检查任务管理器（CPU/内存）
3. 重启 V18，已完成的阶段会从断点继续

### Q: 输出的文书有错别字或格式问题
A: AI 偶尔会出错。V18 的 5 道门禁会捕获明显的错别字，但**专业审核仍由律师负责**。编辑后即可使用。

### Q: 想调整文书模板
A: 高级功能：设置 → 模板管理。可上传自己的 DOCX 模板（带 Jinja2 占位符），V18 会用你的模板填充。
